Encoding Test Server

Hello! This is a http encoding test-server. You can send a http POST request to this url with a Content-Encoding=<input-charset> and a Accept-Encoding=<output-charset> and it will convert the received data to the new format. Here is an example curl request:

        curl -H "Accept-Charset: shift-jis" \
             -H "Content-Encoding: utf-8" \
             -s --data "こんにちは！" \
             localhost:8080

This request will convert the input string from utf-8 to shift-jis
